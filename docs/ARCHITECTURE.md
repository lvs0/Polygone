## Data Flow

### Send Flow
1. `cmd_send()` вҶ’ Parse arguments, load keys
2. `Session::new_initiator()` вҶ’ ML-KEM encapsulation
3. `session.establish()` вҶ’ Derive topology + keys
4. `session.send()` вҶ’ Encrypt + fragment
5. `p2p::build_swarm()` вҶ’ Connect to network
6. DHT PUT вҶ’ Store fragments (30s TTL)

### Receive Flow
1. `cmd_receive()` вҶ’ Load secret key
2. `Session::new_responder()` вҶ’ ML-KEM decapsulation
3. `session.establish()` вҶ’ Re-derive keys
4. DHT GET вҶ’ Collect 4 fragments
5. `session.receive()` вҶ’ Reconstruct + decrypt
6. `session.dissolve()` вҶ’ Zeroize all keys
