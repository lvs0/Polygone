# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Disclosure Policy

We take security issues seriously. If you discover a vulnerability, please report it responsibly.

**How to report:**
- Email: **polygone@proton.me**
- Please include a description of the vulnerability, steps to reproduce, and any relevant details.
- We aim to respond within 48 hours and will keep you informed of our progress.
- We request that you give us reasonable time to address the issue before disclosing it publicly.

**What to expect:**
- We will acknowledge receipt of your report promptly.
- We will work to verify and understand the scope of the issue.
- We will release a fix and public disclosure as appropriate.

Thank you for helping keep Polygone secure.
